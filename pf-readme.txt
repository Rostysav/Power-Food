1. Для того щоб завантажити файл, в папці, де буде розміщений проект, відкрити термінал
   і в ньому ввести команду git clone https://github.com/Rostysav/Power-Food.git
2. В папці появиться папка з назвою проекту, потрібно вже відкрити термінал в папці проекту.
3. Зробити в терміналі наступне: git init, git checkout -b "name of your branch"
4. Потім в терміналі зробити npm install
5. Для відкриття проекту, потрібно ввести ng serve
